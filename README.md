# Cloudburst Prediction System (SIH 2025)

A hardware-integrated cloudburst prediction system using IoT weather sensors and machine learning. It processes real-time rainfall, humidity, temperature, and wind data to estimate cloudburst risk and provide early disaster-management alerts.
